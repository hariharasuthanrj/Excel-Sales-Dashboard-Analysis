# Excel-Sales-Dashboard-Analysis

**Project Overview:**

Data becomes more valuable when it is transformed into meaningful insights. In this project, I used Microsoft Excel to clean, organize, analyze, and visualize customer purchasing data by building an interactive dashboard.

The objective was to convert raw sales data into an easy-to-understand dashboard that helps identify customer buying patterns based on demographic and lifestyle factors. Using Pivot Tables, Pivot Charts, Slicers, and Excel functions, I created a dynamic dashboard that allows users to explore the data through interactive filters.

This project was inspired by Alex The Analyst's Excel Dashboard tutorial, where I recreated the dashboard while understanding the purpose behind each step and the business questions being answered.

**Project Objectives:**

This project focuses on transforming raw customer data into an interactive dashboard by:

Cleaning and preparing the dataset

Creating meaningful calculated fields

Summarizing data using Pivot Tables

Building interactive visualizations

Enabling dynamic filtering with Slicers

Presenting business insights in a user-friendly dashboard
- - - 

**Workflow**

1. Data Preparation

The raw dataset was reviewed and prepared before analysis by:

Checking for missing or inconsistent values

Formatting columns for consistency

Creating helper columns where necessary

Organizing the data for Pivot Table analysis

2. Data Analysis

Used Pivot Tables to summarize customer purchasing behavior based on multiple attributes, including:

Income

Age Group

Commute Distance

Gender

Region

3. Dashboard Development

An interactive dashboard was built using Pivot Charts and Slicers to allow users to filter the analysis dynamically.

The dashboard provides a clear overview of customer purchasing trends and enables quick comparisons across different demographic groups.
